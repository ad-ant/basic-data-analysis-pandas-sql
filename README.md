# Basic Sales Data Analysis

This project presents a basic exploratory data analysis of sales data using
Python, Pandas and SQL.  
It was created as a portfolio project demonstrating fundamental skills required
for a Data & AI Internship role.

## Scope
- Loading and exploring tabular sales data
- Time-based aggregations using Pandas
- Basic data grouping and summarization
- SQL analysis using SQLite within a Jupyter Notebook
- Drawing simple business conclusions from data

## Technologies
- Python
- Pandas
- SQLite
- Jupyter Notebook

## Project Structure

## Key Insights
- Sales show monthly variation across the year
- Electronics products generate the highest revenue
- Poland and Germany are the strongest markets by sales value

## Notes
The project focuses on clarity and correctness rather than complexity.
It reflects a minimal but complete data analysis workflow.
